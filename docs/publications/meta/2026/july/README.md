## Description

Meta announced an open-source MCP server and new API endpoints for FBDL, making it possible for security researchers to generate, validate, submit, monitor, and archive FBDL runs from their own AI-agent workflows.

The article notes that the project began as an open-source MCP server Ads built during the 2026 Meta Bug Bounty Researcher Conference. Meta extended that work with API integration, token management, and full run lifecycle support so researchers can use their own tools, such as Claude Code, OpenAI Codex, or other MCP-compatible agents, to build test environments.
