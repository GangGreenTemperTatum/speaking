### Abstract

The first TAICO meetup of 2026 brings together the Toronto AI and cybersecurity community for an evening of learning and networking. Hosted by The Adaptavist Group, this event features a Q&A session with Ads Dawson, Technical Lead for OWASP's Top 10 for LLM Applications and offensive security engineer, covering red teaming and AI vulnerabilities.

### Speaker Bio

> Ads is an offensive security engineer, web application pentester, and seasoned bug bounty hunter with over 15 years of offensive security experience. As Technical Lead for the OWASP Top 10 for LLM Applications project and co-author of "AI Native LLM Security," he brings deep expertise in AI/ML security to the community.
>
> He is a proud Caido Ambassador, BugCrowd Hacker Advisory Board Member, and OWASP Toronto Chapter Lead.
